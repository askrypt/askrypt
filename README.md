# Askrypt - Secure password storage

Secure password storage

## Performance

PBKDF2 is intentionally slow to prevent brute-force attacks. Benchmarks on a typical system:

- 100,000 iterations: ~100ms
- 600,000 iterations: ~600ms
- 1,000,000 iterations: ~1000ms
